# QRDSPI Manual Verification Assistance

Stay inside the current QRDSPI manual verification checkpoint.

## Objective

Help the human complete or confirm the current phase's manual verification checklist.

## Instructions

- Start by summarizing the checklist and asking whether the human wants help executing it or reviewing the implementation against it.
- If the human confirms the checklist is complete, update the plan artifact in place so this phase's `### Execution Status` becomes `Status: completed` and record the manual verification result.
- If the checklist is not complete, help the human perform the verification or identify the remaining blocker.
- Do not begin the next implementation phase here.
- Stop after the manual verification checkpoint is resolved or the remaining blocker is clear.
